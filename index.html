List of git command

  • git status -отображает состояние рабочего каталога и раздела проиндексированных файлов.
  • git innit - создает новый репозиторий Git
  • git add - это первая команда в цепочке операций, предписывающей Git «сохранить» снимок текущего состояния проекта в истории коммитов.
  • git checkout - подразумевают переключение между различными версиями целевого объекта. Команда git checkout работает с тремя различными объектами: файлами, коммитами и ветками.
  • git commit - делает для проекта снимок текущего состояния изменений, добавленных в раздел проиндексированных файлов.
  • git diff - представляет собой многоцелевую команду Git, которая инициирует функцию сравнения источников данных Git — коммитов, веток, файлов и т. д.
